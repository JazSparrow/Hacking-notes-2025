# what's a net cat?

## Description

Using netcat (nc) is going to be pretty important. Can you connect to `jupiter.challenges.picoctf.org` at port `64287` to get the flag?

##  1 Hints 

nc [tutorial](https://linux.die.net/man/1/nc)

